Installation of Joomla on your web server.

Install and Configure Joomla on Your LAMP Server - Debian

Difficulty: 1
Time: 5

Joomla is an open source content management system that helps you build websites and online applications. Joomla is built in PHP and uses a database system such as MySQL.

This article walks you through the installation of Joomla on your Debian server. Before getting started, you need to have LAMP (Linux, Apache, MySQL, PHP) set up on your server. For instructions, see Build a LAMP stack (Linux, Apache, MySQL, PHP) - Debian.

Download Joomla

  1. Before you download the installation files, create a directory.
    sudo mkdir temp
  2. Go to that directory.
    cd temp
  3. Download the latest version of Joomla from the Joomla website.
    sudo wget http://joomlacode.org/gf/download/frsrelease/17410/76021/Joomla_2.5.7-Stable-Full_Package.tar.gz
  4. Unzip the downloaded package, moving it to the default Apache directory.
    sudo tar zxvf Joomla_2.5.7-Stable-Full_Package.tar.gz -C /var/www
  5. If the command tar is not found, install tar.
    sudo apt-get install tar

Grant permissions

  • Alter the permission on the configuration file. You can change it back once the installation is complete.
    sudo touch /var/www/configuration.php
    sudo chmod 777 /var/www/configuration.php

Create a Joomla database and user

  1. Log into MySQL.
    sudo mysql -u root -p
  2. When prompted, enter the MySQL root password that you set during the MySQL installation.
  3. Create a database with any name.
    CREATE DATABASE testdb;
    The following displays: 
    Query OK, 1 row affected (0.00 sec)
  4. Create a user.
    CREATE USER newuser@localhost;
  5. Set the password for the newly created user.
    SET PASSWORD FOR newuser@localhost= PASSWORD("pass");
  6. Grant privileges to the newly created user.
    GRANT ALL PRIVILEGES ON testdb.* TO newuser@localhost IDENTIFIED BY 'pass';
  7. Refresh MySQL.
    FLUSH PRIVILEGES;
  8. Exit the MySQL shell.
    exit
  9. Restart Apache.
    sudo service apache2 restart

Install Joomla

  1. In your browser, navigate to http://server-ip-address/installation. The Joomla installation page displays.
  2. Follow the onscreen instructions for the Pre-installation Check and License.
  3. Configure the database connection settings, and enter the credentials you created in the previous section.
  4. Enter configuration information for your website such as the name of the site, email, and password for Admin. Click Install Sample Data.
    To complete the installation, you are asked to remove the installation folder.
  5. In your console, remove the installation folder, and change the permission on the configuration file.
    sudo rm -rf /var/www/installation/
    sudo chmod 755 /var/www/configuration.php
  6. In your browser, navigate to http://server-ip-address/index.php or the domain name. The new Joomla page displays.

Next steps

Now that you have set up a Joomla site, you can install Joomla Console to manage your Joomla site by command line.
