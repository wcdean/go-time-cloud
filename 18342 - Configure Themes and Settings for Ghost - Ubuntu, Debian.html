Configure Themes and Settings for Ghost - Debian, Ubuntu

Difficulty: 2

Time: 30 minutes

Ghost is an open-source project for bloggers. It provides a simple blogging platform, is fast and easy to set up, and provides good customization options. This article helps you set up a Ghost blogging platform using NGINX on Debian or Ubuntu. The theme used is available as freeware from the Ghost Marketplace.

Prerequisites

  • Install and configure Ghost with NGINX - Ubuntu
  • Install and configure Ghost with NGINX - Debian

Change the Ghost Theme

  1. If already not running, start the Ghost server.
    cd /home/ghost
    sudo NODE_ENV=production forever start index.js
  2. Log in and then move to the themes directory of Ghost.
    cd /home/ghost/content/themes/
  3. Install Git and git-core unless they are already installed.
    sudo apt-get update && sudo apt-get install git-core -y
  4. Clone the N-Coded theme (as an example) from the GitHub project.
    sudo git clone https://github.com/polygonix/N-Coded.git

Change the Ghost settings

  1. Launch the Ghost setup page on your browser, by typing following URL: http://your_IP_address/ghost/setup
  2. Click on the CREATE YOUR ACCOUNT button to start with Ghost setup
  3. Provide information about email address, name, password and blog title.
  4. Provide names of your team members to invite them on Ghost platform, or skip this step by selecting I'll do this later option.

    Once setup is complete, Ghost CMS Content tab will be displayed.



  5. Go to the general settings page, select the theme from the Theme dropdown list, and click Save.

Conclusion

In this article, we reviewed the steps to configure themes and settings for Ghost on Debian or Ubuntu.
