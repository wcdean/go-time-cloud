Install and configure PostgreSQL - Ubuntu

Difficulty: 1
Time: 15 minutes

This article describes installation and configuration of PostgreSQL on Ubuntu. PostgreSQL is an open-source relational database management system. PostgreSQL uses the Structured Query Language (SQL) to manipulate data.

To perform these tasks, we assume that you have sudo privileges.

Install PostgreSQL

  1. Update all the system packages.
    sudo apt-get update
  2. Install the PostgreSQL package and necessary dependencies.
    sudo apt-get install postgresql postgresql-contrib
  3. Configure PostgreSQL to start up upon server boot.
    sudo update-rc.d postgresql enable
  4. Start the PostgreSQL service.
    sudo service postgresql start
    Once PostgreSQL installs completely, the postgres system user and the database are automatically created.

Connect to PostgreSQL

  1. Log in with the postgres user.
    sudo su - postgres
  2. Create a database user.
    createuser -s username
  3. Log into the database.
    psql
  4. Set the password for database user.
    \password username
  5. Create a database.
    create database databasename;
  6. Switch to the database you just created.
    \connect databasename;
  7. Create a table in the database.
    CREATE TABLE Table_name (name varchar(80), location char(100));
  8. List all the objects.
    \d+
  9. Drop a table.
    DROP TABLE Table_name;
  10. Remove or drop the database.
    drop database databasename;
  11. Exit from the postgres=# prompt.
    \q
  12. Log out the postgres system user.
    exit
