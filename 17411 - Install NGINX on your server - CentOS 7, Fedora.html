Install Nginx on Your Server - CentOS 7, Fedora

Difficulty: 1
Time: 5 minutes

This article guides you through installing and starting Nginx on CentOS 7 and Fedora 20, 21.

Install and start Nginx

  1. Install Nginx:
    sudo yum install nginx
    Enter y at the prompt to continue installation.
  2. Start the Nginx service:
    sudo systemctl start nginx

Nginx is now running as a web server. To verify, you can use your browser to navigate to the IP address of your cloud server; for example, http://IP Address of your cloudserver .
