Install and configure concrete5 on your LAMP server - Debian, Ubuntu

Difficulty: 1

Time: 30 minutes

Introduction

Concrete5 is an open-source content management system (CMS) made in PHP. Concrete5 is easy to understand and learn. In this article you will learn, how to install and configure concrete5 on your LAMP server - Debian and Ubuntu.

Before you proceed, you must have access to sudo privileges to perform the steps mentioned in this article. In addition, you will need to build a LAMP stack by referring the following links:

  • Build a LAMP stack (Linux, Apache, MySQL, PHP) - Debian
  • Build a LAMP stack (Linux, Apache, MySQL, PHP) - Ubuntu

Install the required dependencies

  1. Install the unzip and the PHP GD libraries.
    sudo apt-get install unzip php5-gd
  2. Restart the Apache services to make the changes effective.
    sudo service apache2 restart

Basic configuration

  1. Navigate to the Apache document root:

    Debian:

    cd /var/www/

    Ubuntu:

    cd /var/www/html
  2. In a browser, visit the official Concrete5 downloads page.
  3. Copy the download link for the newest version of Concrete5.
  4. Use this link with wget to download the latest version as a zip file:
    sudo wget --trust-server-names https://www.concrete5.org/download_file/-/view/82243/8497/ -O concrete5.zip
  5. Extract the downloaded source.
    sudo unzip concrete5.zip
  6. The folder will be named concrete5.x.x.x with each number representing the version number. Use the mv command to simplify this directory.
    sudo mv concrete5.x.x.x concrete5
  7. Navigate to the concrete5 directory.
    cd concrete5
  8. Enable the rewrite module.
    sudo a2enmod rewrite
  9. Open your php.ini file in a text editor.
    sudo vim /etc/php5/apache2/php.ini
  10. Uncomment the following line from the file.
    cgi.fix_pathinfo=1
  11. Save and close the file.
  12. Restart the Apache server.
    sudo service apache2 restart

Create database

  1. Create a blank database for backend operation. Log into MySQL.
    mysql -u root -p
  2. When prompted enter mysql password and create database.
    CREATE DATABASE concrete5;
  3. Create the database administrator account, and set the user's credentials. We'll use "admin" as the example user name, and "adminpwd" as the password.
    CREATE USER admin@localhost IDENTIFIED BY 'adminpwd';
    4. As a best security practice, create a strong password. Keep your user name and password handy, as you'll need them later.
  4. Give, or grant, your admin permissions to the database.
    GRANT ALL PRIVILEGES ON concrete5.* TO admin@localhost;
  5. GRANT is one of the account management statements that the MySQL server notices and loads into memory right away. Therefore, a manual reload/restart should not be necessary. However, if you like, you can check the permissions using SHOW GRANTS:
    SHOW GRANTS FOR 'admin'@'localhost';
  6. If the privileges don't take, flush the privileges to activate the user. FLUSH forces a reload of the cache.
    FLUSH PRIVILEGES;
  7. Exit MySQL to return to your normal command prompt.
    exit

Grant Permissions

  1. Change ownership of the document root to make it writable by Concrete5.
    sudo chown -R www-data:www-data /var/www/html/concrete5

    2. Remember that if you are using Debian, you will use /var/www/ in your path instead of /var/www/html.

Install Concrete5

  1. In your web browser, type in the public IP or URL of your concrete5 server as http://server_ip/concrete5, and then press the Enter key.
  2. Make sure that all server requirements are met, and click Continue to Installation.
  3. Enter the required information for Site, Database, and Administrator. Use the MySQL credentials that you created earlier for the Database Information section. Unless you are using a remote database server, your Server field should just be localhost.
  4. Select the option to install sample content, or create an empty site.
  5. Once all fields have been filled, click Install concrete5.
  6. After the installation process is complete, click Continue to your site.
    You will be redirected to the Welcome screen of concrete5.
  7. Select any of the guides to get started.

Conclusion

In this article, you have learned to install and configure concrete5 on your LAMP server.
