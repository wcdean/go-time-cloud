Getting started with installing Git on Arch Linux.

Install Git - Arch Linux

Difficulty: 1
Time: 5 minutes

Git is an open source, distributed version control system (DVCS) used by software developers to handle Linux kernel development projects of all sizes with speed and efficiency. In a DVCS, every developer clones a copy of all versions of a project’s files and has the full history of the project on their hard drive.

Commonly used for source code management, each Git working directory is its own repository, with a complete history and version tracking. The most popular DVCS is GitHub.

This tutorial will walk you through how to install Git on an Arch Linux system. The steps are for a non-root user with sudo (administrator) privileges. For more on how to set up a non-root user with sudo for Arch Linux, see Initial Server Setup - Arch Linux.

Install Git using Pacman

This guide will go over how to install Git from your package manager. Note that this method will get you the current version of Git available on your distribution.

  1. Use this one-line command to install Git:
    sudo pacman -Sy git
  2. At the prompt, Proceed with installation?, enter "y," and then press Enter to finish installation.

If the command completes without error, Git has downloaded and installed.
