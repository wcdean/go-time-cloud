Compile Node.js With NPM From Source - Ubuntu, Debian

Difficulty: 3
Time: 20 minutes

Node.js is a server-side javascript runtime environment. Stable versions are available for install via standard package managers such as apt (Ubuntu, Debian) and yum (Fedora, CentOS). However, installing from source may be preferred in environments where current or beta versions are not available. In this article, we will install the current version of Node.js from a source package.

Step 1: Install Dependency Packages and Tools

  1. Update the local package database
    sudo apt-get update
  2. Install make, gcc and other relevant tools and packages
    sudo apt-get install -y make g++ libssl-dev git

Step 2: Download Source

  1. Navigate to the 'opt' directory
    cd /opt
  2. Download latest source version
    sudo wget http://nodejs.org/dist/node-latest.tar.gz
  3. Extract the source
    sudo tar xzvf node-latest.tar.gz
  4. Navigate to the folder created during the source extraction. At the time of this writing, the latest node source version available was node-v4.2.1.
    cd node-v*

Step 3: Configure, Build and Install

  1. Run the configuration script
    sudo ./configure
  2. Compile the code. NOTE: This command will take a while to complete.
    sudo make
  3. Install Node.js
    sudo make install
  4. Verify Node.js installation
    node --version
    The output should be similar to the following: 
    v2.14.7
  5. Verify npm installation
    npm --version
    The output should be similar to the following: 
    4.2.1

Conclusion

In this article, you compiled, configured and installed the current version of Node.js from source.
