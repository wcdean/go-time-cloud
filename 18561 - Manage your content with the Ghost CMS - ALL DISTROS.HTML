Manage Your Content with Ghost CMS - ALL DISTROS

Difficulty: 3
Time: 1.5 hrs

Ghost is a powerful blogging platform. It's not only simple to set up, but its user interface lets you manage your content with ease. This article provides an introduction to various content management options provided by Ghost.

The tasks in this article assume that the host machine has a Ghost blogging platform deployed and running. For instructions, see one of the following:

  • Install and configure Ghost with NGINX - Ubuntu
  • Install and configure Ghost with NGINX - CentOS, Fedora
  • Install and configure Ghost with NGINX - Arch Linux
  • Install and configure Ghost with NGINX - Debian

Ensure that Ghost is running

  • In a browser, type the IP address or domain name: http://192.169.170.227:2368 or http://coolexample.com.

Start a new blog with Ghost

  1. Open the blog setup page in the browser: http://192.169.170.227:2368/ghost/setup/
  2. Provide the required details, and then click on the LAST STEP: INVITE YOUR TEAM button.

    Then it will prompt you to invite your team. You can skip this step by clicking on I'll do this later take me to my blog. After the registration is complete, the Content tab shows the sample preconfigured blog as Welcome to Ghost.

Create a new post

  1. Click NEW POST on the top left corner.


  2. Enter the post title and content in the fields provided. While you enter the content, a preview of the post displays in the PREVIEW window.
  3. If you need help formatting the content, click the small icon on the bottom right of the Markdown window (the content editing window). To format content, Ghost uses its own markup language called Markdown.

    The Markdown Help window shows additional formatting options, such as bold, emphasize, and link.

    The option for adding tags is in settings section of the sidebar

    The options to publish the draft, save the draft, or delete the post are in the top right corner.

    To access additional post settings such as post URL, publish dates, author name, and other metadata, use the Settings icon, just before SAVE DRAFT button in the top right corner.

Additional blog settings and information

The Settings tab provides additional blog settings:

  • General: General information about the blog such as title, description, logo, cover images, email address, number of posts per page, permanent links with dates, and blog themes.
  • Users: List of active users of the blog.
  • Tags: Tags applied across all the articles in the blog.
  • Navigation: Additional navigation options besides Home.
  • Code Injection: Adds external code into the top and bottom of the blog theme, without editing the theme files.
  • Labs: Testing ground for experimental features.
  • About: Information about the Ghost version, environment, database, mail, and other issues.
Additional help options are available via the question mark icon next to the User Name at the bottom left.

Conclusion

In this article, you learned various ways to manage your content using Ghost. The Ghost blogging platform provides easy-to-use options to manage your content. Because most options are self-explanatory, you can create, edit, publish, and manage posts without additional training.
