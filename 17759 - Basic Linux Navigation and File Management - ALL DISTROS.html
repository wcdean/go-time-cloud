Basic Linux Navigation and File Management - ALL DISTROS

Difficulty: 1
Time: 15 minutes

In this article, we will assist you to understand the basic Linux navigation and file management in all distros. To perform this task, we are assuming that the user has sudo privileges.

Find where you are in a directory

  • To find the present location of any directory, type:
    pwd
    The result should look like this:
    /home/username

Look at the contents of directories

Use the ls command to view the contents of directories.

  • To change the directory, type:
    cd /usr/share
  • To list all the directories and files, type:
    ls
    The output will be shown as:
  • To find all of the information about permissions of all directories and files, type:
    ls -l

  • To get a listing of all files, including hidden files and directories, type:
    ls -al

View files

  • To view a specific file, type:
    less /etc/services
    The output will be shown as:
    • To scroll in the output file, press the up arrow and down arrow keys on the keyboard.
    • To scroll one page up the screen, press either the Ctrl-B or Page Up keys on the keyboard.
    • To scroll one page down the screen, press either Control/Ctrl+F, the spacebar, or the Page Down key on the keyboard.
  • To search something like mail in this file, type:
    /mail
  • To exit the file, type:
    :q
  • To display the content of a particular file, type:
    cat file1
  • To display top 10 lines of a particular file, type:
    head /etc/services
  • To display last 10 lines of a particular file, type:
    tail /etc/services

Work with files and directories

There are several commands that can create files and directories.

  • To create an empty file with the name and location that you specify, type.
    sudo touch file1
  • To create multiple files at the same time, type:
    sudo touch file1 file2 file3 file4
  • To create a directory, type:
    sudo mkdir test
  • To create nested directories, type:
    sudo mkdir -p /test/share/example
  • To rename directories and file, type either:
    sudo mv test share
    sudo mv file1 example1
  • To copy directories and files, type:
    sudo cp -rvf /test again
    For example:
    sudo cp –rvf /etc/services /usr/file1
    • -r means recursive
    • -v means visible
    • -f means force
  • To remove files directories, type the appropriate command:
    sudo rm –rvf /etc/services.txt
    sudo rm –rvf /etc/test
    • -r means recursive
    • -v means visible
    • -f means forcedly

Edit files

  1. Open a file, type:
    sudo vim /etc/services
    • To go into insert mode and make changes whatever you want, press the I key.
    • To go into maintenance mode, press ESC.
    • To save the file, type:
      :wq!
  3. To exit without saving the file, type:
    :q!

Conclusion
