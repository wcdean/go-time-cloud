Install UFW firewall to increase security on Debian and Ubuntu

Install an Uncomplicated Firewall (UFW) - Debian, Ubuntu

Difficulty: 1
Time: 5 minutes

Uncomplicated Firewall (UFW) is the front end for IPtables. Its command-line user interface makes it easy to configure your Linux™ firewall. Ubuntu and some Debian systems come prepackaged with UFW.

This article walks you through the basics of installing and enabling UFW.

Install UFW

  1. Update your packages:
    sudo apt-get update
    It's always good practice to update your packages before installing anything new.
  2. Install UFW.
    sudo apt-get install ufw
  3. (Debian only) If you get an error message about missing dependencies (such as "E: Unable to fetch some archives, maybe run apt-get update or try with --fix-missing?"), update your package manager.
    sudo apt-get install --fix-missing

Enable UFW

  1. Check the status of UFW to make sure it's not already running.
    sudo ufw status
  2. View the output:
    • If the output is similar to ufw start/running, the firewall is active. You're finished!
    • If the firewall is not active, enable UFW.
      sudo ufw enable
      Enter y to at the prompt, "Command may disrupt existing ssh connections. Proceed with operation (y|n)?"
      The system returns this output:
      Firewall is active and enabled on system startup

Next steps

Congratulations! You installed and/or enabled UFW. From here you can do either of the following:

  • Set up your firewall (by managing connections and rules). See Set up UFW firewall - Debian, Ubuntu.
  • Learn more on how to use UFW with IPv6. See Use UFW firewall with IPv6 - Debian, Ubuntu.
