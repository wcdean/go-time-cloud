Install Ruby on Rails using RVM - Debian

Difficulty: 2
Time: 20 minutes

Ruby on Rails™ is a well-known framework for web application development. The Ruby Version Manager (RVM) allows developers to easily manage multiple Ruby application environments and switch between them in an efficient way.

This article walks through how to install Ruby on Rails on Debian using RVM, and assumes that you have sudo privileges. See Initial Server Setup - Debian for more on how to set up your server.

Prepare your system

Before you install Ruby on Rails, prepare your system with the required system software.
  1. Update the system software and access root.
    sudo apt-get update
  2. Install curl.
    sudo apt-get install curl
  3. Install tar.
    sudo apt-get install tar
  4. Install all the dependencies.
    sudo apt-get install libgdbm-dev libncurses5-dev automake libtool bison libffi-dev

Update and install RVM and Ruby

  1. Install rvm dependencies and rvm.
    sudo curl -sSL https://rvm.io/mpapis.asc | gpg --import -
    sudo curl -L https://get.rvm.io | bash -s stable
  2. To apply RVM, you need to execute the following command:
    source ~/.rvm/scripts/rvm
  3. Install Ruby and check Ruby version using following commands:
    rvm install ruby
    ruby -v
  4. Ruby installs the contents for all the packages. To inhibit this, type the following command:
    sudo echo "gem: --no-ri --no-rdoc" > ~/.gemrc
  5. Install nodejs using apt-get.
    sudo apt-get install nodejs

Install common and useful gems

Bundler is a tool to manage gems while developing and running Rails applications. Nokogiri is a dependency for many other gems.

  1. Update the gem manager.
    gem update --system
  2. Type the following command to use Ruby with gemset global:
    rvm gemset use global
  3. Install both of the gems.
    gem install bundler
    gem install nokogiri

Install Rails

  1. Use gem to install Rails. This will take some time to install.
    gem install rails
    To check the version, type:
    rails -v

Next steps

Congratulations! You've set up Ruby on Rails using RVM for your web application. For more information on RVM, see Use RVM to manage Ruby installations.
