Check the version of your Linux distribution

Check your Linux version - ALL DISTROS

Difficulty: 1
Time: 5 minutes

There are a number of Linux-based operating systems and, like other software, each version has a unique identifier. The version number provides important information about the operating system, including its date of release, its capabilities, and duration of its official support. A simple command provides version information for a number of different operating systems (or "distributions").

Check your version information

  • In a terminal window, enter the following command:
    $ cat /etc/*-release
    Each system returns similar information:
    OS Output
    Arch Linux
    • NAME shows that the operating system (or Linux distribution) is Arch Linux.
    CentOS
    • NAME shows that the operating system (or Linux distribution) is CentOS Linux.
    • VERSION shows the version and code name. "7 (Core)" indicates the seventh release with a code name of "Core."
    Debian
    • NAME shows that the operating system (or Linux distribution) is Debian GNU/Linux.
    • VERSION shows the version and code name. "7 (wheezy)" indicates the seventh release with a code name of "wheezy." (Trivia fact: Wheezy comes from a character in Disney Pixar's Toy Story.)
    Fedora
    • NAME shows that the operating system (or Linux distribution) is Fedora.
    • VERSION shows the version and code name. "21 (Twenty One)" indicates the twenty-first release with a code name of "Twenty One." (Trivia fact: Fedora's code names are not as interesting as Debian's.)
    Ubuntu
    • DISTRIB_ID shows that the operating system (or Linux distribution) is Ubuntu.
    • VERSION shows the complete version and code name. "14.04.2 LTS, Trusty Tahr" indicates the second version that was released in October 2014. The "LTS" designator indicates that the version is eligible for long-term support, so it is officially supported for five years after release. The code name for this particular release is "Trusty Tahr."

Conclusion

Now you know how to determine which version of the operating system is running on your server.
