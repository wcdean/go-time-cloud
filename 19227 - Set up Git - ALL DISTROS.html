Getting started with setting up (configuring) Git on all distros

Set Up Git - ALL DISTROS

Difficulty: 1
Time: 5 minutes

After installing Git, you can set up your Git user name and email address. Every Git commit uses these details. Without them, you get an error message (and have to revise) each comment you make.

Before you begin, you'll need to have Git installed. Choose your system for instructions:

  • Install Git - Arch Linux
  • Install Git - CentOS, Fedora
  • Install Git - Debian
  • Install Git - Ubuntu

Set up your Git user name and email address

Git comes with a configuration tool called git config that defines how it looks and works.

  1. Specify your user name and email address. Replace the user name and user email with your own:
    git config --global user.name "your username"
    git config --global user.email youremail@yourdomain.com
  2. Check your settings.
    git config --list
    Git lists all the settings it can find. The output looks like this:
    user.name=your username
    user.email=youremail@yourdomain.com
