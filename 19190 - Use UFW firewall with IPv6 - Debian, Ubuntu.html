Set up UFW firewall to work with IPv6

Use UFW firewall with IPv6 - Debian, Ubuntu

Difficulty: 1
Time: 5 minutes

Internet Protocol Version 6 (IPv6) is the standard network-layer protocol supporting the Internet and is the followup version to IPv4 (also referred to as "IP"). Ubuntu and Debian come with all the tools you need to use IPv6. This article walks you through how to use Uncomplicated Firewall (UFW) with IPv6 as well as IPV4.

First, make sure that you have UFW installed. See Install a UFW firewall - Debian, Ubuntu for instructions.

Set up UFW to work with IPv6

  1. Open the UFW configuration file.
    sudo vim /etc/default/ufw
  2. Look for this section, which should be near the top:
    # Set to yes to apply rules to support IPv6 (no means only IPv6 on loopback
    # accepted). You will need to 'disable' and then 'enable' the firewall for
    # the changes to take affect.
    IPV6=yes
  3. Make sure that it includes IPV6=yes.
  4. Save and close the file.
  5. Reload UFW by disabling it first and then enabling it:
    sudo ufw disable
    sudo ufw enable
  6. Enter y at the prompt to continue.
    The output should show:
    Firewall is active and enabled on system startup
That's it! UFW now configures both IPv4 and IPv6 firewall rules when needed.

Next steps

You have set up UFW to work with IPv6. If you haven't already, you can also set up the UFW firewall by allowing/denying connections and defining rules.
