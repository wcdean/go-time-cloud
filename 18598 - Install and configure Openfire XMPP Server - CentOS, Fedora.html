Install and Configure Openfire XMPP Server - CentOS, Fedora

Difficulty: 2
Time: 0.5 hours

This article will help you to understand the installation and configuration of Openfire XMPP server on CentOS or Fedora.

Extensible Messaging and Presence Protocol (XMPP) is an XML-based communication protocol for middleware. It enables the near real-time exchange of structured yet extensible data between any two or more network entities.

Openfire enables collaboration in real time and provides a tightly secured and high performing server. You can use the open source Apache license for Openfire and can easily set up and configure it using Java.

Openfire is a Java-based tool to use the utility of XMPP server.

Openfire disallows chat options.

Install Openfire

  1. Update the operating system.
    sudo yum –y update
  2. Install Java.
    sudo yum install java
  3. Download Openfire.
    sudo wget http://www.igniterealtime.org/downloadServlet?filename=openfire/openfire-3.9.3-1.i386.rpm -O /root/openfire-3.9.3-1.i386.rpm
  4. Install Openfire.
    sudo yum install -y /root/openfire-3.9.3-1.i386.rpm

    The Openfire package does not require separate installation of Java Runtime Environment (JRE) as it provides its own. This package is available in 32-bit version only. So, if you are running this installation on a 64-bit machine, corresponding 32-bit libraries must be installed separately.

  5. Install 32-bit libraries (if you use a 64-bit system).
    sudo yum install -y glibc.i686
  6. Start the service again after every reboot.
    sudo chkconfig openfire on
  7. Start the service.
    sudo systemctl start openfire.service

Test Openfire

  1. Test the Openfire installation on your system by opening the following URL on your browser: http://your_IP:9090.
    Successful installation of Openfire shows the Openfire wizard on your browser.
  2. Follow the instructions to proceed with the setup.
    • On the first page, choose the language.
    • On the second page, provide appropriate values for domain and associated ports.
    • Keep the default values for the rest of the steps.
  3. After the setup procedure is complete, log into the Administration Console using the admin account.

    After you log in, the first screen shows the Server Manager tab under the Server tab. The Server Manager provides information about server properties and environment, system properties, language and time, clustering, database, logs, email settings, and the security audit viewer.

    The Server Settings tab provides information about profile settings, client connections, server to server connections, external components, HTTP bindings, update management, registration and login management, resource policies, offline messages, audit policies for messages, private data storage options, security settings, server certificates, compression settings, and file transfer settings.

    The Media Services tab provides information about media proxy settings.

Conclusion

By using the above steps, you can install and configure Openfire XMPP server on CentOS or Fedora.
