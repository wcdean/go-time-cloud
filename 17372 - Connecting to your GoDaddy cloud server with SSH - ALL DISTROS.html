Connect to Your Cloud Server with SSH - ALL DISTROS

Difficulty: 1
Time: 10 minutes

Once you create a COMPANY_NAME cloud server, you need to connect to it before you can manage it. In this article, you'll connect through SSH using OpenSSH (which runs on Linux or Mac OS X) or PuTTy (which runs on Windows). SSH, which stands for Secure Shell, is a protocol for secure access to a remote computer.

To connect to your cloud server using one of these tools, you need the following:

  • User name: The server's user with privileges to connect remotely. This user name can be the same name used in the Security Info section when creating the server.
  • Password: Password entered when creating your user. The password helps to authenticate the user.
  • Server IP address: This address is how you find your server on the Internet. This address is available in the Servers section of the COMPANY_NAME servers page.

Connect to your server with OpenSSH (Linux or Mac OS X)

OpenSSH is a command line tool.

  1. Open a terminal window.
  2. Start an OpenSSH connection:
    ssh username@serverIPaddress
    The first time you attempt to connect, a security warning displays. The server is not recognized because you haven't connected to it before.
  3. Type yes to continue to open the connection. The connection is added to the list of hosts in your computer.
  4. At the prompt, enter your credentials. Once your user name and password are authenticated, you are connected.

Connect to your server with PuTTY (Windows)

PuTTY is one of the most popular SSH clients for Windows. You can download it for free from the PuTTy download page.

  1. Double-click putty.exe to start it. The configuration screen where you enter the server details displays.
  2. Enter the connection settings:
    • Host Name (or IP address): The name (or the IP address) of the server you wish to connect to.
    • Port: The port to use for the connection. The default is 22.
    • Connection Type: The type of connection you want to use. SSH is the default.
  3. Open the connection: Click Open and accept the security alert. (This alert displays only the first time you connect to the server)
  4. At the authentication prompt, enter your user name and password.
    Once the credentials are valididated, you are connected.
