Configure Themes and Settings for Ghost - CentOS, Fedora

Difficulty: 2

Time: 30 minutes

Ghost is an open-source project for bloggers. It provides a simple blogging platform, is fast and easy to set up, and provides good customization options. This article helps you set up a Ghost blogging platform using Nginx on CentOS or Fedora. The theme used is available as freeware from the Ghost Marketplace.

Before starting, make sure to install and configure Ghost with Nginx.

Change the Ghost theme

  1. Log in, and then move to the themes directory of Ghost.
    cd /var/www/ghost/content/themes/
  2. Install Git and NodeJS package manager unless they are already installed.
    sudo yum -y install git wget unzip man
    sudo yum install npm
  3. Clone the N-Coded theme (as an example) from the GitHub project.
    sudo git clone https://github.com/polygonix/N-Coded.git
  4. Change the ownership of the Ghost user.
    sudo chown -R apache:apache N-Coded

    If you receive an invalid user error, create a Ghost user and then retry.

  5. Restart npm production services.
    sudo npm start --production

Change the Ghost settings

  1. In a browser, type http://your_IP_address/ghost/settings/general in the address bar.
    You are redirected to the time setting page.
  2. Fill in the required information about your blog and credentials on the setup form.
  3. Click OK, LET'S DO THIS.
  4. In the general settings page, select the theme from the Theme dropdown list, and click Save.
  5. Refresh your Ghost public home page to see the blog with the updated theme.

Conclusion

In this article, you learned to configure themes and settings for Ghost on CentOS or Fedora.
