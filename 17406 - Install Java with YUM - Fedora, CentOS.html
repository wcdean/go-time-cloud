Install Java

Install Java with yum - CentOS, Fedora

Difficulty: 1
Time: 10 minutes

Java is used by many applications, so it behooves many administrators to have Java already installed. This article contains instructions for installing the Java runtime SDK and Open JDK.

Install Java

  1. Update your repository.
    sudo yum update
  2. Find the latest versions of Java in your package manager.
    yum search jdk
    Note the version you want.
  3. Download Java runtime using this format for your command.
    sudo yum install java-version number-openjdk
    Replace the version number with the version you want. For example, we'll download version 1.7.0.
    sudo yum install java-1.7.0-openjdk
    Enter y at the prompt to continue installation.
  4. Download Java OPEN JDK 7. Do the same for the SDK.
    sudo yum install java-version number-openjdk-devel
    Again, we'll download version 1.7.0.
    sudo yum install java-1.7.0-openjdk-devel
    Enter y at the prompt to continue installation.
  5. You can also get the latest version number and tar.gz archive from the Oracle downloads page. This site requires a license agreement.

Check your Java version

  • Check the versions of Java installed on your server.
    java -version
    If your output shows just one version of Java installed, as in this example, you are all finished!
    java version "1.7.0_79"
    OpenJDK Runtime Environment (fedora-2.5.5.0.fc20-x86_64 u79-b14)
    OpenJDK 64-Bit Server VM (build 24.79-b02, mixed mode)
    If you have more than one Java version installed, continue on to select and verify the default version.

(Optional) Select and verify the default Java version

If you have multiple versions of Java on the same server, choose your default version. Then set your environment.

Select the default version of Java

  1. If you have more than one version of Java installed, you can use the following command to select it. Unless you installed an Oracle version above, this command will only return one row.
    sudo update-alternatives --config java
    You will see this: 
    Selection   Command
-----------------------------------------------
*+ 1      /usr/lib/jvm/java-1.7.0-openjdk-1.7.0.79-2.5.5.0.fc20.x86_64jre/bin/java

Enter to keep the current selection[+], or type selection number:
  2. Create a /etc/environment file.
    sudo vim /etc/environment
  3. To select your preferred version, use the JAVA_HOME command and the path of your preferred alternative with the jre/bin/java part of the path removed. Enter this path into the file.
    JAVA_HOME="/usr/lib/jvm/java-1.7.0-openjdk-1.7.0.79-2.5.5.0.fc20.x86_64"
  4. Save and close.
  5. Load the environment file.
    source /etc/environment
  6. Test it.
    echo $JAVA_HOME

Conclusion

Congratulations! You have succesfully installed Java on your server.
