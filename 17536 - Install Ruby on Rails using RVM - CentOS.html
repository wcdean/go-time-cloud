Ruby on Rails™ is a well-known framework for web application development. The Ruby Version Manager (RVM) allows developers to easily manage multiple Ruby application environments and switch between them efficiently.

This article will walk through how to install Ruby on Rails on CentOS 6 or 7 using RVM, and assumes that you have sudo privileges. See Initial Server Setup - CentOS for more on how to set up your server.

Prepare your system

Before you install Ruby on Rails, prepare your system with the required system software.
  1. Update the system software and access root by running the following command:
    sudo yum update
  2. Install curl.
    sudo yum install curl
  3. Install tar.
    sudo yum install tar

Update and install RVM and Ruby

  1. Install the RVM dependencies using the following commands:
    sudo curl -sSL https://rvm.io/mpapis.asc | gpg2 --import -
  2. If your system doesn't have RVM already installed, type the following command to install RVM:
    sudo curl -L https://get.rvm.io | bash -s stable
  3. To apply RVM, execute the following command:
    source ~/.rvm/scripts/rvm
    If your system already has RVM installed, then update it.
    rvm get stable --autolibs=enable

Install Ruby

  1. Use RVM to install Ruby.
    rvm install ruby
  2. To check Ruby version, type:
    ruby -v

  3. The Asset Pipeline in Rails needs JavaScript runtime to compile code. On CentOS, it's best to use Node.js (a server-side JavaScript environment).

    Before installing nodejs, first add it to your PPA repository:
    sudo yum install nodejs

Install common and useful gems

Bundler is a tool to manage gems while developing and running Rails applications. Nokogiri is a dependency for many other gems.

  1. Update the gem manager.
    gem update --system
  2. Type the following command to use Ruby with gemset global:
    rvm gemset use global
  3. Install both the gems.
    gem install bundler
    gem install nokogiri

Install Rails

  1. Use gem to install the most recent stable release of Rails.
    gem install rails
    To check the version, type:
    rails -v

Next steps

Congratulations! You've set up Ruby on Rails using RVM for your web application. For more information on RVM, see Use RVM to manage Ruby installations.
