Install Java with pacman - Arch Linux

Difficulty: 1
Time: 10 minutes

Java is used by many applications, so it benefits many administrators to have Java already installed. This article contains directions for installing both the Java runtime and SDK using the Arch Linux pacman package deployment tool.

Install Java

  1. Update your repository.
    sudo pacman -Syu
  2. Find the current versions of Java available in your package manager.
    pacman -Ss openjdk
    Note the version you want.
  3. Download Java runtime using this format for your command.
    sudo pacman -S jreversion number-openjdk
    Replace the version number with the version you want. For example, we'll download version 8.
    sudo pacman -S jre8-openjdk
    Enter y at the prompt to continue installation.
  4. Download Java OPEN JDK using this format. Do the same for the SDK.
    sudo pacman -S jdkversion number-openjdk
    Again, we'll download version 8.
    sudo pacman -S jdk8-openjdk
    Enter y at the prompt to continue installation.
  5. You can also get the latest version number and tar.gz archive from the Oracle downloads page. This site requires a license agreement.

Check your Java version

  • Check the versions of Java installed on your server.
    java -version
    If your output shows just one version of Java installed, as in this example, you are all finished!
    openjdk version "1.8.0_60"
    OpenJDK Runtime Environment (build 1.8.0_60-b24)
    OpenJDK 64-Bit Server VM (build 25.60-b23, mixed mode)
    If you have more than one Java version installed, continue on to select and verify the default version.

(Optional) Select and verify the default Java version

If you have multiple versions of Java on the same server, choose your default version. Then set your environment.

Select the default version of Java

  • Create or edit the file /etc/environment.
    sudo vim /etc/environment
  • Select your Java_home. In this case, we will use the jdk version.
    JAVA_HOME="/usr/lib/jdk8-openjdk"
  • Save and close.
  • Load the environment file.
    source /etc/environment
  • Test it.
    echo $JAVA_HOME

    • Conclusion

    Congratulations! You have succesfully installed Java on your server.
