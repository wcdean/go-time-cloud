Install Apache, MySQL, and PHP on a server running Ubuntu to serve web applications.

Build a LAMP (Linux, Apache, MySQL, PHP) stack on a server running Ubuntu

Difficulty: 3
Time: 20 minutes

To build a dynamic web application, you need what has been coined a “stack” – which is developer lingo for an integrated set of software that has all of the components your application needs.

Most stacks have the same types of component, and the stacks differ in which pieces of software they use for those components. For example, we’ll use LAMP (which this article covers):

Component LAMP Software
Operating system Linux (e.g. Ubuntu)
Web server Apache
Database MySQL
Programming language PHP

If you're astute, maybe you noticed that LAMP is just an acronym for the software the stack uses. We'll walk you through installing all of these (except for Linux, which is already installed as your OS when you create the server).

Install Apache

  1. Before you begin any installation, make sure that you update the repository apt:
    sudo apt-get update
  2. Install Apache:
    sudo apt-get install apache2
  3. To verify that Apache is installed, type this URL in a browser:
    http://your server's IP address.
    If Apache is installed, the Apache Test Page displays.

    COMPANY_NAME Cloud Server customers can use

    .

Install MySQL

  1. Install MySQL with a few helper packages:
    sudo apt-get install mysql-server php5-mysql
  2. You will be asked to set a MySQL root password. If you skip this step, you can configure the password later.
  3. Create the database directory structure for MySQL:
    sudo mysql_install_db
  4. Run the following security script:
    sudo mysql_secure_installation
  5. You will be asked for the root password, which you set earlier. If you didn't set the password, or if you want to change it, you can do so now. If you want to continue with the current password, type "N" at the prompt.
  6. You will be asked more questions as part of security configuration. It is a best practice to respond "Y" to these system prompts.

Install PHP

  1. To install PHP, open the terminal and type:
    sudo apt-get install php5 libapache2-mod-php5 php5-mcrypt
  2. In order to configure how Apache serves its files, you can make some changes. Open the dir.conf file:
    sudo vim /etc/apache2/mods-enabled/dir.conf
  3. Move the index.php file to the first position after Directory Index.
  4. Restart Apache:
    sudo service apache2 restart

Install PHP modules

If your applications requires any PHP modules, you can install them now.

  1. View available PHP modules:
    apt-cache search php5-
  2. The list of available PHP modules displays. If you want to get more information on a particular package, you can display the description by typing:
    apt-cache show the name of the package you want to use
  3. Install the package you want:
    sudo apt-get install the name of the package you want to use

Test PHP processing on Apache

  1. Create a new PHP file under the /var/www/html directory:
    sudo vim /var/www/html/info.php
  2. After the file opens, type in the following code:
    <?php
    phpinfo();
    ?>
  3. Save and close the file:
    :wq!
  4. To verify it works, type this URL in your browser: http://your server's IP address/info.php

A page displays with the PHP version, extensions, build date, and other information.

Next steps

Now that you have a LAMP stack, you can use it for dynamic, database-driven websites powered by things like WordPress or Drupal.
