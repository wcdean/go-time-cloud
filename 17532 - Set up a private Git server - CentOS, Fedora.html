Set up git server.

Set up a private Git server - Centos, Fedora

Difficulty: 2
Time: 10 minutes

Even in the open-source world, sometimes you need a little privacy.  Or maybe you don't want to expose your code tot the world until you have cleaned it up a bit.  In these cases, it may be prefereable to set up a private Git server.

Preparation

  1. Create a public/private RSA key pair.
    ssh-keygen -C "your email address"
    Follow the directions to create a key. We will name the key "mygitkey".
  2. Add a git user
    sudo useradd git
  3. Add a password for the git user
    sudo passwd git

Install git

  1. Install git
    sudo yum install git
  2. Create a .ssh directory for key storage
    sudo mkdir -p /home/git/.ssh
  3. Copy your rsa public key to an authorized key file.
    sudo cp mygitkey.pub /home/git/.ssh/authorized_keys
  4. Create a directory for your git project(s).
    sudo mkdir -p /usr/gitprojects
  5. Navigate to your git project directory.
    cd /usr/gitprojects
  6. Create a new project
    git init --bare my-cool-project.git

Next steps

Your git server is set-up with a new project. Visit the git download website to download a git client and connect it to your new repository. You may also want to review "Using Git to automate deployment to your server"
