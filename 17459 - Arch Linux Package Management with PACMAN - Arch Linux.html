Arch Linux Package Management with PACMAN - Arch Linux

Difficulty: 2
Time: Time value

PACMAN is one of the utility tools for package management in Arch Linux. Using this tool, you can install, update, remove, query, and search the package from the repository available by Arch Linux.

Install package

Description Command
Install a package online. sudo pacman -S package_name
Install a package that has a similar name. sudo pacman -S lua-{sec,socket,zlib}
Download the package from the repository for archive purposes. sudo pacman -Sw package_name

Update and upgrade

Description Command
Ensure that the package manager is up to date. sudo pacman -Sy
Upgrade the packages. sudo pacman -Su
Upgrade a particular package. sudo pacman -Sy package_name

Search package

Description Command
Search a package from PACMAN repository. pacman -Ss package_name
Search a package with color highlighted output. pacsearch package_name
Search a package using wildcard operator ^. This command will show a list of packages that begin only with the specified text, in this "ruby". pacsearch ^ruby
Guarantee that a regular expression works as intended; surround it with quotes. pacsearch 'python'
List all installed packages using the grep command, to include one line before and after each match. pacsearch linux | grep installed -A 1 -B 1
Check an installed package in noncolored output. pacman -Qs python

Retrieve information

Description Command
Fetch the information about the installed package. pacman -Qi python
Find the size of the installed package. pacman -Qi | grep -e "Name" -e "Package Installed Size"
Print the recently update information of the installed package. pacman -Si python
Print the list of files associated with a package. pacman -Ql vim
Print the list of files without the package name. pacman -Qlq vim
Get the URL of the downloaded package. sudo pacman -U package_path

Remove/uninstall a package

Description Command
Remove or uninstall a package. sudo pacman -R package_name
Remove a package, provided nothing is dependent on it. Remove its dependencies that aren't required by anything else. sudo pacman -Rs package_name
Forcefully remove a package. This is the command to just wipe the package from your system and reinstall, when reinstalling alone isn't enough. sudo pacman -Rdd package_name

Conclusion

In this article, you learned various commands for package management with PACMAN in Arch Linux.
