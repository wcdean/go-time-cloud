Add and delete users on your server - CentOS, Fedora

Difficulty: 1
Time: 10 mins

In this article you will learn to add and delete users on a server running CentOS or Fedora.

Add a new user

  1. Type in the following command:
    sudo adduser username
    Replace username with the name of the user you want to create.
  2. Set the user's password.
    sudo passwd username
    3. Replace username with the user you just created.
  3. Enter and confirm the user's new password.

You have now created a new user with a password.

If you are logged in as root, to add a new user you can use the above commands without the prefix sudo.

Grant sudo access to users

If you want to grant sudo access (administrative privileges) to your users, you can do so by adding the user to the wheel group with the gpasswd command.

sudo gpasswd -a username wheel
Replace username with the user you want to grant sudo access.

If you are logged in as root, you can use the above command without the prefix sudo.

Switch users

If you want to login as the new user you just created, you can use the su command.

su - username
Replace username with the user record you want to delete.

This allows you to switch to the new user without logging out first. If you want to logout before logging in as the new user, you can use the exit command.

Delete an existing user

Type in the following command:

sudo userdel -r username
Replace username with the user you want to delete.

You have now deleted the user record and the user's home directory.

If you are logged in as root, to delete a user you can use the same command userdel -r username without the prefix sudo.
