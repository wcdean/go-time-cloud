Install and configure PostgreSQL - CentOS 7

Difficulty: 1

Time: 15 minutes

PostgreSQL is an open-source, relational database management system. PostgreSQL uses Structured Query Language (SQL) to manipulate data. To perform this task, we are assuming that the user has sudo privileges.

This article provides information about installation and configuration of PostgreSQL on CentOS 7.

Install PostgreSQL

  1. Update all the packages of the operating system.
    sudo yum -y update
  2. Install the PostgreSQL package and its necessary dependencies.
    sudo yum -y install postgresql postgresql-libs postgresql-plperl postgresql-plpython postgresql-server
  3. Initialize the database and data directories.
    sudo service postgresql initdb
  4. Configure PostgreSQL to start up on server boot.
    sudo chkconfig postgresql on
  5. Start the PostgreSQL service.
    sudo service postgresql start

Connect to PostgreSQL

  1. After PostgreSQL is installed completely, a Postgres system user and database will be automatically created.

  2. Log in as Postgres user.
    sudo su - postgres
  3. Log in into database using the following command:
    psql
  4. You will see the Postgres=# prompt. To exit from the prompt, type:
    \q
  5. Log out as the Postgres system user.
    exit
