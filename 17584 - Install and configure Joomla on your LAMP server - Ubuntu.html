Installation of Joomla on your web server.

Install and configure Joomla on your LAMP server - Ubuntu

Difficulty: 1
Time: 10 minutes

Joomla is an open source content management system that helps you build websites and online applications. Joomla is built in PHP and makes use of a database system like MySQL.

This article walks you through the installation of Joomla on your Ubuntu server.

Prerequisite

Before getting started, you need to have LAMP (Linux, Apache, MySQL, PHP) set up on your server. For instructions, see installing LAMP stack.

Download Joomla

  1. Download the latest version of Joomla.
    sudo wget https://github.com/joomla/joomla-cms/archive/master.zip
  2. Install unzip and unzip the downloaded package, moving it to the default Apache directory.
    sudo apt-get install unzip
    unzip -q master.zip
  3. Move the downloaded package to the default Apache directory.
    sudo mv joomla-cms-master/* /var/www/html
  4. Remove the master file.
    rm -rf master.zip

Grant permissions

Change ownership of the directory.
sudo chown -R www-data:www-data /var/www/html

Create Joomla DB and User

  1. Login to MySQL.
    sudo mysql -u root -p
    When prompted, enter the MySQL root password that you set during the MySQL installation.
  2. Create a database with any name.
    CREATE DATABASE testdb;
  3. Create a user.
    CREATE USER joomlauser@localhost;
  4. Set the password for the newly created user.
    SET PASSWORD FOR joomlauser@localhost= PASSWORD("testdbpwd");
  5. Grant privileges to the newly created user.
    GRANT ALL PRIVILEGES ON testdb.* TO joomlauser@localhost IDENTIFIED BY 'testdbpwd';
  6. Refresh MySQL.
    FLUSH PRIVILEGES;
  7. Exit the MySQL shell.
    exit

Configure Apache Install Joomla

  1. In your browser, navigate to http://your server address or domain name or http://your server address or domain name/installation . You will see the Joomla installation page.
  2. Enter Configuration information for your website, such as the name of the site, email and password for the administrator.
  3. Click Next to enter Database configuration details. Enter the new user credentials (joomlauser, testdbpwd) that you created for the testdb database.
  4. Click Next.We will not be entering any information in the FTP Configuration page as we will not be using FTP right now, so simply click Next.
  5. Install Sample Data of any choice or you can choose None.
  6. To complete the installation you will be asked to remove the installation folder. You can delete the installation directory manually from your console.
    sudo rm -rf /var/www/html/installation/
  7. In your browser, navigate to http://your server address or domain name/index.php. The new Joomla page displays.

Configure Apache

  1. You can configure the document root for Apache to use index.php, so that when you browse to your IP address or domain you are directed to the Joomla installation page.
    sudo vim /etc/apache2/mods-enabled/dir.conf
    Move the index.php to the first position after Directory Index, so it looks something like
    DirectoryIndex index.php index.html index.cgi index.pl index.xhtml index.htm
  2. Point your IP address to the domain name so that you can see your Joomla site when you go to http
  3. Restart Apache.
    sudo service apache2 restart
  4. In your browser, navigate to http://yourdomainname and you will see your Joomla site.

Next steps

Now that you have set up a Joomla site, you can install Joomla Console to manage your Joomla site by command line.
