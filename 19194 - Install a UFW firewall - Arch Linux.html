Install UFW firewall to increase security on Arch Linux

Install a UFW firewall - Arch Linux

Difficulty: 1
Time: 5 minutes

Uncomplicated Firewall (UFW) is the front end for Iptables. Its command-line user interface makes it easy to configure your Linux™ firewall. UFW is available in the Arch Linux repositories.

This article walks you through the basics of installing and enabling UFW.

Install Iptables

UFW won't function properly without Iptables, which comes pre-installed in Arch Linux but needs to be set up. See Set up a Firewall using Iptables - ALL DISTROS for instructions.

Install UFW

  1. First, update your packages. It's always good practice to update your packages before installing anything new.
    sudo pacman -Syu
  2. Enter y at the prompt to continue.
  3. Install UFW using pacman.
    4. sudo pacman -Syu ufw
  4. Enter y at the prompt to continue.

Enable UFW

  • Enable UFW.
    sudo ufw enable
    To check the status of UFW:
    sudo ufw status

Next steps

Congratulations! You installed and/or enabled UFW. From here you can do either of the following:

  • Set up your firewall (by managing connections and rules). See Set up UFW firewall - Arch Linux.
  • Learn more on how to use UFW with IPv6. See Use UFW firewall with IPv6 - Arch Linux.
