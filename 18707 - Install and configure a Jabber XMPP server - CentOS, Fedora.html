Setting up Jabber XMPP Server on CentOS and Fedora, XMPP

Install and configure Jabber XMPP server - CentOS, Fedora

Difficulty: 2
Time: 30 Minutes

This article explains the installation and configuration of Ejabber on CentOS and Fedora.

Jabber is a simple, mature, and powerful messaging protocol that was developed to collaborate in real-time between two or more people, based on the text typed. Jabber is used by enterprises for internal communication, developers for programming skills improvement, hosting providers for high availability and stable communication services, and by individuals for chatting.

You will need access to sudo privileges to complete all steps.

Download the Ejabberd installer

  1. Run the following command to download Ejabber:
    sudo wget http://www.process-one.net/downloads/downloads-action.php?file=/ejabberd/14.12/ejabberd-14.12-linux-x86_64-installer.run -O ejabberd-14.12-linux-x86_64-installer.run
  2. Change permissions.
    sudo chmod +x ejabberd-14.12-linux-x86_64-installer.run
  3. Execute the installer.
    sudo ./ejabberd-14.12-linux-x86_64-installer.run

Set up EJabberd

  1. After starting the installer, it will ask for language. Select English language on Number [2].
  2. Read and accept the end user license agreement for the Ejabberd installer by pressing y.
  3. Press Enter to select the following installation directory:
    Installation Directory [/opt/ejabberd-14.12]:

    If you want to keep the default directory, then press Enter. Otherwise, you can choose a different path.

  4. To select the domain, type the fully qualified domain name (FQDN), public IP, localhost, or any internal IP from which we can access the domain.
    ejabberd server domain [domain_name-centos]

Set up administrator credentials

  1. When prompted for the username, enter the username of your administrator.
    Administrator username [admin]:

  2. At the prompt for a password, enter a carefully chosen password.

  3. After submitting the password, installer will prompt for cluster. It is not required, press n to continue.
    promoted massage example Cluster [y/N]:N
  4. To finalize the installation, press y. The installer will start the installation process. It will take some time to complete.
    Do you want to continue?
  5. It will prompt you to view the Readme file. Press y to view it, or n to continue.
    View Readme file? [Y/n]: n

Start the Ejabberd services

  1. Ejabber protocol is ready to be used. Use the following command to start the Jabber protocol:
    cd /opt/ejabberd-14.12/bin/
  2. Create admin user using the following command.
    sudo ./ejabberdctl register admin your-server_ip userpassword
  3. Start the ejabberd service using the following command.
    sudo ./ejabberdctl start
  4. Verify if the Ejabber serviced has been started by using the following command:
    sudo netstat -plunt | grep 5280
  5. If you get the output as shown below, then you have successfully installed the Jabber instant messaging service.
    tcp 0 0 0.0.0.0:5280 0.0.0.0:* LISTEN 27744/beam.smp
  6. Open your browser and type http://your_ip:5280/admin and press Enter. It will prompt for authentication credentials. Enter the credentials that you entered during installation.

    After successfully logging in, you will see this the screen.

Conclusion

You have successuflly installed and setup Ejabberd.
