Install Webmin - Arch Linux

Difficulty: 2
Time: 15 minutes

Webmin is a web-based tool for managing your server. It supports many common tasks like installing software, adding users, and managing sites and applications. This article describes how to install Webmin.

Webmin does not have native support for Arch Linux's Pacman software utility. You will not be able to install software packages, such as Apache or MySQL, directly through Webmin. If you want to be able to use Webmin to install and manage services, you should consider a traditional Linux distribution, such as CentOS/Fedora or Ubuntu/Debian builds.

Get Webmin

  1. Change to the tmp directory.
    cd /tmp
  2. Use Pacman to install a few tools we need for Webmin setup.
    sudo pacman -Syu unzip tar perl-crypt-ssleay
  3. Download Webmin.
    sudo wget http://www.webmin.com/download/webmin-current.tar.gz
  4. Extract the tar archive
    sudo tar -xvf webmin-current.tar.gz
  5. This will create a directory with the current version name.
    [user@server tmp]$ ls
    user_firstboot webmin-1.770 webmin-current.tar.gz
  6. Navigate to that directory.
    cd webmin-1.770
  7. Install Webmin.
    sudo ./setup.sh /usr/local/webmin

    The install program will ask several questions. Accept the defaults by pressing Enter on your keyboard at each one, until you get to the Login name prompt. Create a login name and password. Enter y at the last prompt.

Verification

  1. In your browser, go to the Webmin login page. https://Your domain or IP address:10000
  2. Ignore the self-signed SSL warning and proceed. You can trust this server because you own it.
  3. A login screen will appear. Log in with the username and password you set up in the previous steps, and start using Webmin!
