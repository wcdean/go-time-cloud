Install Oracle Java

Manually Install Oracle Java - ALL DISTROS

Difficulty: 2
Time: 15 minutes

Java is a popular software package available in open source and Oracle (proprietary) flavors. This article describes how to install the Oracle version.

Install Oracle Java

  1. Navigate to the Oracle downloads web site, and get the latest version appropriate to your machine. Copy the URL, paste it into a wget statement as shown below, and run it.
    wget --header "Cookie: oraclelicense=accept-securebackup-cookie" http://download.oracle.com/otn-pub/java/jdk/8u5-b13/jdk-8u5-linux-x64.tar.gz
  2. Make a Java directory.
    sudo mkdir /opt/jdk
  3. Extract the downloaded files into your Java directory.
    sudo tar -zxf jdk-8u5-linux-x64.tar.gz -C /opt/jdk
  4. Install Java Runtime (JRE).
    sudo update-alternatives --install /usr/bin/java java /opt/jdk/jdk1.8.0_05/bin/java 100
  5. Install Java compiler (SDK).
    sudo update-alternatives --install /usr/bin/javac javac /opt/jdk/jdk1.8.0_05/bin/javac 100
  6. Choose your Java default.
    sudo update-alternatives --config java
    Your screen displays the following (or more rows if you have other versions of Java installed). Pick your default version.

Next steps

When the next version of Oracle Java is released, you can repeat this process and use update-alternatives to select the new version and delete the old one if you like.
