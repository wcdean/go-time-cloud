Install Ruby on Rails using rbenv - Ubuntu

Difficulty: 2

Time: 30 minutes

Ruby is a pure object oriented language like PERL and Python, while Rails is a web application development framework written in Ruby. This article will assist you to install Ruby on Rails on Ubuntu, with the help of rbenv. rbenv is a new, lightweight Ruby version management tool.

You'll need sudo privileges for this tutorial. See Initial Server Setup - Ubuntu for more instructions. Optionally, you can also install Git.

Install rbenv

  1. Update all the packages of operating system.
    sudo apt-get update
  2. Install the rbenv and all Ruby dependencies.
    sudo apt-get install git-core curl zlib1g-dev build-essential libssl-dev libreadline-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt1-dev libcurl4-openssl-dev python-software-properties libffi-dev
  3. Create a clone for rbenv from GitHub, and install it.
    git clone https://github.com/sstephenson/rbenv.git ~/.rbenv
  4. Add a directory to your $PATH for accessing rbenv.
    echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc
  5. Add rbenv init into the bashbc shell.
    echo 'eval "$(rbenv init -)"' >> ~/.bashrc
    source ~/.bashrc
  6. Restart the shell by either opening a new tab or closing the current terminal and starting it again.
  7. Check rbenv installed correctly using the following command.
    type rbenv
    If it was set up correctly, your output will include a message saying "rbenv is a function."
  8. Install the latest development version of ruby-build into the ~/.rbenv/plugins/ruby-build directory.
    git clone https://github.com/sstephenson/ruby-build.git ~/.rbenv/plugins/ruby-build
  9. Install the rbenv rehash plugin to automate running the rehash command.
    git clone https://github.com/sstephenson/rbenv-gem-rehash.git ~/.rbenv/plugins/rbenv-gem-rehash

Install Ruby

  1. To see all the available versions of Ruby, use this command:
    rbenv install --list
  2. The format of the command to install Ruby is:
    rbenv install latest_version of Ruby -k
    For instance, to install v2.2.3:
    rbenv install -v 2.2.3
  3. Set the default version of Ruby that all of your shells will use.
    rbenv global latest_version of Ruby
  4. Check your version of Ruby.
    ruby -v

Install RubyGems

  1. By default, when we install gems, documentation files will also get installed along with it. However, this takes time. You can stop the document installation by disabling them.
    echo "gem: --no-document" > ~/.gemrc
  2. After you have disabled the document installation, install the bundler gem. The Bundler gem is an essential tool for managing gems when developing and running Rails applications.
    gem install bundler

Install the JavaScript Runtime

There are a few Rails features like the Asset Pipeline, that requires a Javascript runtime to execute. JavaScript runtime can be installed in a couple of ways. In this article, we will use nodejs from Debian package system.

  1. Add the node.js PPA to apt-get:
    sudo add-apt-repository ppa:chris-lea/node.js
  2. Install the node.js package.
    sudo apt-get install nodejs

Install Rails

  1. To install Rails, type:
    gem install rails

    This will take some time, so be patience. Once completed, Ruby on Rails will be installed.

  2. To install shims for all Ruby executables known to rbenv, use the rehash command.
    rbenv rehash
  3. To check the version of both Ruby and Rails, type:
    ruby -v
    rails -v
