Manage SSH windows

Use Screen to Manage SSH Windows - CentOS, Fedora

Difficulty: 1
Time: 10

Screen is an application that allows you to create and manage multiple terminal windows from a single terminal sessions. In this article, we'll install screen, view the available keyboard shortcuts, and create and switch among multiple windows.

Install screen

  • Run the following commands:
    sudo yum update
    sudo yum install screen

Start screen

  1. Open the screen application:
    screen

Create and manage multiple windows

Understanding how to create and manage multiple windows is a fundamental aspect of using screen. Before we create a new window with screen, let's identify our current window.

  1. Start screen.
  2. At the current shell prompt, type the following:
    Window 0

To create a new window in screen, do the following:

  1. Hold down the Control/Ctrl key, press A, and then release Control/Ctrl.
    listens for key bindings.
  2. Press the C key to create a new window.
  3. At this shell prompt, type the following:
    Window 1

You should now have two windows available: one with the text "Window 0" at the prompt, the other with "Window 1" at the prompt.

To view a list of windows open in screen:

  1. Hold down the Control/Ctrl key, press A, and then release Control/Ctrl.
    screen listens for key bindings.
  2. Hold down the Shift key, and press the " key.
    You should see a list of screen windows:
    Num Name                                                           Flags

   0 bash                                                               $
   1 bash                                                               $

To navigate the list and select a particular window:

  1. Press the Up and Down arrow keys to highlight the line "0 bash".
  2. Press Return/Enter.
    You should see the shell prompt with "Window 0" at the prompt.
  3. Press Control/Ctrl+A " to display the window list again.
  4. Press the Up and Down arrow keys to highlight the line "1 bash".
  5. Press Return/Enter.
    You should see the shell prompt with "Window 1" at the prompt.

To close a single window session:

  1. Using the steps above, select and enter the window you want to close.
  2. Hold down the Control/Ctrl key, press A, and then release Control/Ctrl.
    screen listens for key bindings.
  3. Press the K key.
  4. At the prompt "Really kill this window [y/n]", press the Y key.

To close all window sessions opened via screen:

  1. Using the steps above, select and enter the window you want to close.
  2. Hold down the Control/Ctrl key, press A , and then release Control/Ctrl.
    screen     listens for key bindings.
  3. Press the \ key.
  4. At the prompt, "Really quit and kill all your windows [y/n]", press the Y key.

Conclusion

In this article, you installed screen, viewed a list of available keyboard shortcuts, created new windows, and closed windows created in screen.
