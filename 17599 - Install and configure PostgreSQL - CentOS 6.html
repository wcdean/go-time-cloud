Install and configure PostgreSQL - CentOS 6

Difficulty: 1
Time: 15 minutes

This article describes the installation and configuration of PostgreSQL on CentOS 6. PostgreSQL is an open-source relational database management system. PostgreSQL uses the Structured Query Language (SQL) to manipulate data.

The tasks in this article assume that you have sudo privileges.

Install PostgreSQL

  1. Update all the system packages.
    sudo yum –y update
  2. Install the PostgreSQL package and its necessary dependencies.
    sudo yum -y install postgresql postgresql-libs postgresql-plperl postgresql-plpython postgresql-server
  3. Initialize the database and data directories.
    sudo service postgresql initdb
  4. Configure PostgreSQL to start up upon server boot.
    sudo chkconfig postgresql on
  5. Start the PostgreSQL service.
    sudo service postgresql start
    Once PostgreSQL installs completely, the Postgres system user and database are automatically created.

Connect to PostgreSQL

  1. Log in by Postgres user.
    sudo -i -u postgres
  2. Log into the database.
    psql
  3. Exit from the Postgres=# prompt.
    \q
  4. Log out the Postgres system user.
    exit
