Install Ruby on Rails with rbenv - Debian

Difficulty: 2

Time: 30 mins

Ruby is a pure object-oriented language like Perl and Python. Rails is a framework for web application development written in Ruby. In this article, you'll learn to install Ruby on Rails on Debian with the help of rbenv, a lightweight Ruby version management tool.

This tutorial will walk you through installation on a Debian system. Make sure you have sudo privileges.

To set up the Debian server, refer to Initial Server Setup - Debian and Install Git - Debian.

Install rbenv

  1. Update the Debian package.
    sudo apt-get update
  2. Create a clone for rbenv from GitHub, and install it.
    sudo apt-get install git-core
    git clone https://github.com/sstephenson/rbenv.git ~/.rbenv
  3. Add a directory to your $PATH for accessing rbenv.
    echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc
  4. Add rbenv init into the bashbc shell.
    echo 'eval "$(rbenv init -)"' >> ~/.bashrc
    source ~/.bashrc
  5. Restart the shell by either opening a new tab or closing the current terminal and starting it again.
  6. Check rbenv installed correctly using the following command.
    type rbenv
  7. Install the latest development version of ruby-build into the ~/.rbenv/plugins/ruby-build directory.
    git clone https://github.com/sstephenson/ruby-build.git ~/.rbenv/plugins/ruby-build
  8. Install the rbenv rehash plugin to automate running the rehash command.
    git clone https://github.com/sstephenson/rbenv-gem-rehash.git ~/.rbenv/plugins/rbenv-gem-rehash

Install Ruby

  1. Install the required libraries and dependencies for Ruby.
    sudo apt-get install build-essential libssl-dev libcurl4-openssl-dev libreadline-dev -y
  2. Check the list of available versions of Ruby.
    rbenv install --list
  3. Install Ruby. In this article, we used Ruby version 2.2.3.
    rbenv install -v Latest_ruby_version
  4. Set the default version of Ruby that all of your shells will use.
    rbenv global 2.2.3
  5. Display the version of Ruby that was installed.
    ruby -v

Install JavaScript runtime

A few Rails features depend on a JavaScript runtime.The JavaScript runtime can be installed in a couple of ways. We'll use nodejs from the Debian package system.
  1. Open the file where the Debian repositories are located.
    sudo vim /etc/apt/sources.list
  2. Make sure this is added:
    deb http://ftp.us.debian.org/debian/ wheezy-backports main
    And add this line at the end of the file:
    deb-src http://ftp.us.debian.org/debian/ wheezy-backports main
  3. Save and exit the file.
  4. Install nodejs.
    sudo apt-get -t wheezy-backports install nodejs
    Press Y to continue when prompted.

Install Rails

  1. To install Rails, type:
    gem install rails

    This will take several minutes, so patience is key! Once completed, Ruby on Rails will be installed.

  2. To install shims for all Ruby executables known to rbenv, use the rehash command.
    rbenv rehash
  3. To check the version of Ruby:
    ruby -v
  4. To check the version of Rails:
    rails -v

Next steps
