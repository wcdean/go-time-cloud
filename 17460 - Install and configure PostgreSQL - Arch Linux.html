Install and configure PostgreSQL – Arch Linux

Difficulty: 1
Time: 15 minutes

This article provides information about the installation and configuration of PostgreSQL on Arch Linux.

PostgreSQL is an open-source, relational database management system. PostgreSQL uses the Structured Query Language (SQL) to manipulate data. To perform this task, we are assuming that the user has sudo privileges.

Install PostgreSQL

  1. Update all packages of the operating system.
    sudo pacman -Sy
  2. Install the PostgreSQL package.
    sudo pacman -S postgresql
  3. Install pgadmin3 for the administrator. This is an easy-to-access GUI for PostgreSQL, and is an optional installation.
    sudo pacman -S pgadmin3

    Select the default option.

  4. Check if postgresql is installed.
    sudo pacman -Si postgresql
  5. Login as postgres system user and initialize a database cluster, use the command initdb, which is installed with PostgreSQL
    sudo -i -u postgres
    initdb -D '/var/lib/postgres/data'
  6. You can log out from the postgres system user, and fire up PostgreSQL.
    logout
    sudo systemctl start postgresql
  7. If you want PostgreSQL to start automatically every time your VPS boots up, use this command.
    sudo systemctl enable postgresql

Connect to PostgreSQL

  1. Once PostgreSQL is installed completely, then the Postgres system user and database will be automatically created.
  2. Log in as the Postgres user.
    sudo -i -u postgres
    psql
  3. You will see Postgres=# prompt. To exit from the prompt, type:
    \q
  4. Logout the postgres system user.
    exit
