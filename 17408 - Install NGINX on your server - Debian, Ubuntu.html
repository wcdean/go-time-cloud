Install Nginx on Your Server - Debian, Ubuntu

Difficulty: 1
Time: 5 minutes

This article guides you through installing and starting Nginx on Debian or Ubuntu.

Install and start Nginx

  1. Install Nginx by doing one of the following:
    • Download the Nginx software, find your OS version, and follow the instructions.
    • If the Nginx installation package is not included with your OS (or if you don't know), run these commands:
      sudo apt-get update
      sudo apt-get install nginx
  2. Start the Nginx service:
    sudo service nginx start
    Nginx is now running as a web server. To verify, you can use your browser to navigate to the IP address of your cloud server; for example, http://IP Address of your cloudserver .
