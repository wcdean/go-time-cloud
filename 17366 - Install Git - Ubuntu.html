Install Git – Ubuntu

Difficulty: 1
Time: 5 minutes

Git is an open source, distributed version control system (DVCS) used by software developers to handle Linux kernel development projects of all sizes with speed and efficiency. In a DVCS, every developer clones a copy of all versions of a project’s files and has the full history of the project on their hard drive. Commonly used for source code management, each Git working directory is its own repository, with a complete history and version tracking. The most popular DVCS is GitHub.

This tutorial will walk you through how to install Git on an Ubuntu system. The steps will be for a non-root user with sudo (administrator) privileges. For more on how to set up a non-root user with sudo for Ubuntu, see Initial Server Setup - Ubuntu.

Install Git using apt

This guide will go over how to install Git from your package manager. Note that this method will get you the current version of Git available on your distribution.

Before installing Git, use the apt package management tools to upgrade all your installed packages.

  1. Upgrade your installed packages.
    sudo apt-get update
  2. Install Git.
    sudo apt-get install git
    If the command completes without error, Git has downloaded and installed.
  3. You can also verify the install by checking the version.
    git --version
    If the check produced a Git version number, the installation is complete.
