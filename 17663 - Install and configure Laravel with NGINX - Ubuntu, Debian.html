Instructions to install and configure Laravel in a NGINX / Ubuntu, Debian environment.

Install and configure Laravel with NGINX - Ubuntu, Debian

Difficulty: 3
Time: 20 minutes

Laravel is a framework used to create web applications.

Before you start, be sure you have a Ubuntu or Debian installed. And if your server is 512 MB or smaller, it's recommended to add swap for Laravel to install correctly.

Install dependencies

  1. Update your installation package.
    sudo apt-get update
  2. Get these additional items:
    sudo apt-get install nginx php5-fpm php5-cli php5-mcrypt git

Configure PHP

  1. Enable mcrypt.
    sudo php5enmod mcrypt
  2. Restart PHP.
    sudo service php5-fpm restart

Configure NGINX for Laravel

  1. First, create a physical location for the Laravel server.
    sudo mkdir -p /var/www/laravel
  2. Create a new Laravel configuration file.
    sudo vim /etc/nginx/conf.d/laravel.conf
  3. Add the following contents to the newly-created file: 
    server {
listen       81;
root /var/www/laravel/public;
index index.php index.html index.htm;
server_name localhost;
try_files $uri $uri/ /index.php?$query_string;

# Load configuration files for the default server block.

location / {
        }

        error_page 404 /404.html;
            location = /40x.html {
        }

        error_page 500 502 503 504 /50x.html;
            location = /50x.html {
        }

location ~ \.php$ {
try_files $uri /index.php =404;
fastcgi_split_path_info ^(.+\.php)(/.+)$;
fastcgi_pass unix:/var/run/php5-fpm.sock;
fastcgi_index index.php;
fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name; 
include fastcgi_params;
 }

    }
  4. Save and close the file.

Make your main Nginx config file read your Laravel server block

  1. Open your main Nginx config file to edit.
    sudo vim /etc/nginx/nginx.conf
  2. Make sure this include line is present. It should be under the heading # Virtual Host Configs. Add this server_names line underneath it. 
    include /etc/nginx/conf.d/*.conf;
server_names_hash_bucket_size 64;
  3. Restart Nginx.
    sudo service nginx restart
  4. Restart PHP.
    sudo service php5-fpm restart

Install Composer

Composer is a tool we will use to manage the Laravel implementation.

  1. Navigate to your home directory
    cd ~
  2. Run the Composer install script.
    curl -sS https://getcomposer.org/installer | php
  3. Change the name of the newly-created composer.phar file to "composer"
    sudo mv composer.phar /usr/local/bin/composer

Install Laravel

  1. Use composer to retrieve and install Laravel into your /var/www directory. This might take a few minutes.
    sudo /usr/local/bin/composer create-project laravel/laravel /var/www/laravel 4.2
  2. Change the owner to "www-data."
    sudo chown -R www-data:www-data /var/www/laravel
  3. Grant permissions to your Laravel directory.
    sudo chmod -R 755 /var/www/laravel
  4. Change the permissions to allow www-data to write to the app/storage directory.
    sudo chmod -R 755 /var/www/laravel/app/storage
  5. In your web browser, verify success by navigating to your server, appended by :81.
    your server IP address:81

Conclusion

You have arrived. Now you can use the Laravel framework to make beautiful apps.
