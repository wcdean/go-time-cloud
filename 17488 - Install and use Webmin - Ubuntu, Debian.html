Webmin is a web-based tool for managing your server. It supports many common tasks like installing software, adding users, and managing sites or applications.

Install Webmin - Debian, Ubuntu

Difficulty: 2
Time: 15 minutes

This article describes how to install Webmin.

Install Webmin

  1. Edit the /etc/apt/sources.list file.
    sudo vim /etc/apt/sources.list
  2. Add the following lines to the bottom of the file. This will add the necessary APT repositories for Webmin. 
    deb http://download.webmin.com/download/repository sarge contrib
deb http://webmin.mirror.somersettechsolutions.co.uk/repository sarge contrib
  3. Save and close.
  4. Add the Webmin .asc key to your APT keyring.
    wget -q http://www.webmin.com/jcameron-key.asc -O- | sudo apt-key add -
  5. Update your local APT repository.
    sudo apt-get update
  6. Install Webmin.
    sudo apt-get install webmin
    Enter y at the prompt to continue installation.
  7. At the end of the response from the install process you will see this: 
    ...
Webmin install complete. You can now login to https://YourMachineName:10000/
as root with your root password.
  8. Change your Webmin root password with the command below. Use a secure password. This will not affect your system's root password.
    sudo /usr/share/webmin/changepass.pl /etc/webmin root newpassword

    9. If you used an SSH key to log in to your server, then you will need to manually change this root user. Even if you just used a normal login for your server, it is still a good idea to change this anyway so that your system root user and your Webmin root user do not share the same password.

  9. Restart the Webmin service.
    sudo service webmin restart

Verification

  1. In your browser, go to the Webmin login page. https://Your domain or IP address:10000
  2. Ignore the self-signed SSL warning and proceed. You can trust this server because you own it.
  3. A login screen will appear. Log in using root as your user name and the newpassword you set up in the previous step, and start using Webmin!
