CentOS and Fedora Package Management with RPM and YUM - CentOS, Fedora

Red Hat Package Manager (RPM) and Yellowdog Updater, Modified (YUM) are open source, default package-management utility tools for Red Hat based operating systems like CentOS and Fedora. Using RPM and YUM, you can install, reinstall, uninstall, update, and upgrade packages. RPM and YUM are also used to check the version, status, signature, and dependencies using the RPM and YUM tools. A package consists of an archive of files and metadata used to install and erase the archive files.

This article will help you understand package management using RPM and YUM respectively. We'll be using a variety of package examples (indicated like so: package examples) to illustrate how these commands are used.

RPM

The RPM formerly known as .rpm file includes compiled software programs and libraries needed by the packages. This utility works only with packages that are built on .rpm format.

Description Command
Check the RPM signature package. rpm --checksig http://vault.centos.org/centos/7/extras/Source/SPackages/python-sqlalchemy-0.9.8-1.el7.src.rpm
Install the RPM package. sudo rpm -ivh ftp://fr2.rpmfind.net/linux/centos/7.1.1503/os/x86_64/Packages/nmap-ncat-6.40-4.el7.x86_64.rpm
-i: Install a package.
-v: Verbose the message/output.
-h: Print the hash of the packages.
Check dependencies for installation. sudo rpm -qpR ftp://fr2.rpmfind.net/linux/centos/7.1.1503/os/x86_64/Packages/nmap-ncat-6.40-4.el7.x86_64.rpm
Option description:
-q: Query a package.
-p: List capabilities that this package provides.
-R: List the package dependencies.
Install a RPM package without dependencies. sudo rpm -ivh --nodeps ftp://fr2.rpmfind.net/linux/centos/7.1.1503/os/x86_64/Packages/nmap-6.40-4.el7.x86_64.rpm
Verify the installed RPM package. sudo rpm -Vp ftp://fr2.rpmfind.net/linux/centos/7.1.1503/os/x86_64/Packages/nmap-6.40-4.el7.x86_64.rpm
List the files installed by RPM. sudo rpm -ql python
List the recently installed RPM packages. sudo rpm -qa -last
Print all the installed packages. sudo rpm -qa
Upgrade the RPM packages. sudo rpm upgrade
Uninstall an RPM package. sudo rpm -evv nmap
Query a file that belongs to the RPM package. sudo rpm -qf /usr/bin/ls
Print the information of installed RPM. sudo rpm -qi python
Import the PGP keys to verify the packages. sudo rpm --import /etc/pki/rpm-gpg/RPM-GPG-KEY-CentOS-7

Above example is for CentOS 7.x
List all the imported GPG keys. sudo rpm -qa gpg-pubkey*

YUM

YUM is an automatic update and package-management utility. YUM installs all the dependencies automatically.

Description Command
Update the YUM packages. sudo yum update
Install a package. sudo yum install httpd
Update the installed YUM package. sudo yum update httpd
Search a package from the yum repository. sudo yum search httpd
Remove or uninstall the package. sudo yum erase httpd
or
sudo yum remove httpd
Clear the caches of packages. yum clean all
Check if the packages need to be updated. sudo yum check-update
List a package or groups of packages. sudo yum list
sudo yum list httpd
Print what a package does. yum provides httpd
Find the list of RPM dependencies. sudo yum deplist package-name
Print the list of groups. sudo yum grouplist | more
YUM group install. sudo yum groupinstall 'Development Tools'
Remove or uninstall YUM group. sudo yum groupremove 'Development Tools'
Update YUM group. sudo yum groupupdate 'Development Tools'
Get YUM group information. sudo yum groupinfo 'Development Tools'
Print the list of installed packages in paging. sudo yum list installed | more
