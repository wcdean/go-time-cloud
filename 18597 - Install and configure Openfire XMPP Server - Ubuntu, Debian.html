Install and Configure Openfire XMPP Server - Debian, Ubuntu

Difficulty: 2

Time: 20 minutes

This article will help you to install and configure Openfire XMPP server on Debian or Ubuntu.

Extensible Messaging and Presence Protocol (XMPP) is an XML-based communication protocol for middleware. It enables the near real-time exchange of structured yet extensible data between any two or more network entities.

Openfire enables collaboration in real time and provides a tightly secured and high performing server. You can use the Open Source Apache License for Openfire and can easily set up and configure it using Java.

Openfire is a java based tool.

Openfire disallows chat options.

Install Openfire

  1. Update the operating system.
    sudo apt-get update
  2. Install Java.
    sudo apt-get install default-jre
  3. Download Openfire from the source.
    sudo wget -O openfire.deb http://www.igniterealtime.org/downloadServlet?filename=openfire/openfire_3.8.2_all.deb
    This command initiates the download of the Openfire Debian package to openfire.deb.
  4. Install the openfire.deb package.
    sudo dpkg --install openfire.deb
  5. Start the service.
    sudo /etc/init.d/openfire start

Test Openfire

  1. Visit http://your_IP:9090 in your browser.
    A successful Openfire installation displays the Openfire wizard.
  2. Follow the instructions to continue the setup.
    • On the first page, choose the language.
    • On the second page, provide appropriate values for the domain and associated ports.
    • Keep the default values for the remaining steps.
    • On Final page Account Setting press on Skip This Step without type anything.
  3. After completing the setup procedure, log into the Administration Console using the Admin account.

    After you log in, the first screen shows the Server Manager tab under the Server tab. The Server Manager provides information about server properties and environment, system properties, language and time, clustering, database, logs, email setting and security audit viewer.

    The Server Settings tab provides information about profile settings, client connections, server to server connections, external components, http bindings, update management, registration and login management, resources policies, offline messages, audit policies for messages, private data storage options, security settings, server certificates, compression settings and file transfer settings.

    The Media Services tab provides information about Media Proxy settings.

Conclusion

You have installed and configured Openfire XMPP server on Debian or Ubuntu.
