Install and configure Concrete5 on your LAMP server - Arch Linux

Difficulty: 1
Time: 20 minutes

Introduction

Concrete5 is an open-source content management system (CMS) made in PHP. Concrete5 is easy to understand and learn. In this article you will learn, how to install and configure Concrete5 on your Arch Linux LAMP server.

Before you proceed, you must have access to sudo privileges to perform the steps mentioned in this article. In addition, you will need to build a LAMP stack.

Install the required dependency

  1. Install the unzip tool, which is required to extract the file.
    sudo pacman -S unzip
  2. Install the PHP libraries for graphics.
    sudo pacman -S php-gd

Update the PHP configuration file

  1. Open php.ini to enable various graphics and modules.
    sudo vim /etc/php/php.ini
  2. Uncomment the following lines in the file. 
    extension=mysql.so
extension=mysqli.so
extension=iconv.so
extension=gd.so
  3. Save and close.
  4. Restart the Apache services to make the changes effective.
    sudo systemctl restart httpd.service

Basic configuration

  1. Download the the source of concrete5 from the official site, using the following command:
    cd /srv/http
    sudo wget --trust-server-names http://www.concrete5.org/download_file/-/view/58379/8497/ -O concrete5.zip
  2. Extract the downloaded source.
    sudo unzip concrete5.zip
  3. Rename the file concrete5.6.2.1 as concrete5 to shorten the name.
    sudo mv concrete5.6.2.1 concrete5
  4. Change directory to concrete5.
    cd concrete5

Create database

  1. Access MySQL.
    mysql -u root -p
  2. Create a blank database for backend operation.
    CREATE DATABASE concrete5;
  3. Exit MySQL.
    exit

Grant Permissions

  1. Change ownership of the following items to make them writable for concrete5.
    sudo chown -R http:http /srv/httpconcrete5
  2. Change permissions of the following folders.
    sudo chmod -R 755 /srv/httpconcrete5

Install Concrete5

  1. In your web browser, type in the public IP or URL of your concrete5 server http://server_ip/concrete5, and then press the Enter key.
  2. On the installation page, click Continue to Installation.
  3. Enter the required information for Site, Database, and Administrator, and then click Install concrete5.
  4. After the installation process is complete, click Continue to your site.You will be redirected to the Welcome screen of concrete5.

  5. Select any of the guides to get started.
