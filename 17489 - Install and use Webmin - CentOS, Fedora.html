Install Webmin - CentOS, Fedora

Difficulty: 2
Time: 15

Webmin is a web-based tool for managing your server. It supports many common tasks like installing software, adding users, and managing sites and applications. This article describes how to install webmin on CentOS and Fedora.

  1. Install perl-Net-SSLeay.
    sudo yum install perl-Net-SSLeay
  2. Navigate to your /opt directory
    cd /opt
  3. Download the Webmin key ASC key.
    sudo wget http://www.webmin.com/jcameron-key.asc
  4. Import the ASC key to your RPM keyring.
    sudo rpm --import jcameron-key.asc
  5. Download the Webmin RPM package.
    sudo wget http://www.webmin.com/download/rpm/webmin-current.rpm
  6. Install Webmin.
    sudo rpm -Uvh webmin-*.rpm
    At the end of the response from the install process you will see this: 
    ...
Webmin install complete. You can now login to https://YourMachineName:10000/
as root with your root password.
  7. Change your Webmin root password with the command below. Use a secure password. This will not affect your system's root password.
    sudo /usr/libexec/webmin/changepass.pl /etc/webmin root newpassword

    8. If you used an SSH key to log in to your server, then you will need to manually change this root user. Even if you just used a normal login for your server, it is still a good idea to change this anyway so that your system root user and your Webmin root user do not share the same password.

Verification

  1. In your browser, go to the Webmin login page. https://Your domain or IP address:10000
  2. Ignore the self-signed SSL warning and proceed. You can trust this server because you own it.
  3. A login screen will appear. Log in using root as your user name and the newpassword you set up in the previous step, and start using Webmin!

Next steps
