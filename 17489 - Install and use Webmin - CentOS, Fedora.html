Install Webmin - CentOS, Fedora

Difficulty: 2
Time: 15

Webmin is a web-based tool for managing your server. It supports many common tasks like installing software, adding users, and managing servers. This article describes how to install webmin on CentOS 6, 7 and Fedora 20, 21. It assumes you have a user, yourloginuser, who accesses your server via SSH.

  1. Install perl-Net-SSLeay.
    sudo yum install perl-Net-SSLeay
  2. Navigate to your /opt directory
    cd /opt
  3. Add the Webmin key to your keyring.
    sudo wget http://www.webmin.com/jcameron-key.asc
  4. Import the public key.
    sudo rpm --import jcameron-key.asc
  5. Get Webmin from rpm.
    sudo wget http://www.webmin.com/download/rpm/webmin-current.rpm
  6. Install Webmin.
    sudo rpm -Uvh webmin-*.rpm
    At the end of the response from the install process you will see this: 
    ...
Webmin install complete. You can now login to https://YourMachineName:10000/
as root with your root password.
    7. There are two problems with the response. First, the install responds with your machine name, not your domain name. When you verify the install (see below), you will use your domain name or IP. Second, you do not want to send your root password out over the Internet.
  7. Change your Webmin root password with the command below. This will not affect your system's root password.
    sudo /usr/libexec/webmin/changepass.pl /etc/webmin root newpassword

Verification

  1. In your browser, go to the Webmin login page. https://Your domain or IP address:10000
  2. Ignore the SSL warning and proceed. You can trust this server because you own it.
  3. A login screen will appear. Log in using root as your user name and the newpassword you set up in the previous step, and start using Webmin!
